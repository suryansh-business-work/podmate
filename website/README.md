# PartyWings Website

Landing page for PartyWings platform built with Astro.

## Tech Stack

- Astro
- TypeScript

## Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Scripts

```bash
pnpm dev          # Start dev server (port 4041)
pnpm build        # Production build
pnpm preview      # Preview production build
```
